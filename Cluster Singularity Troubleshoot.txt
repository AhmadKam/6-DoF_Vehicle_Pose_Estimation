TROUBLESHOOT:

- If CUDA_HOME environment variable is not set.

	Set path of Cuda before running command for e.g.:

		CUDA_HOME=/usr/local/cuda-9.2 python setup.py install

- If error in dlopen or dlsym: libcaffe2_nvrtc.s RUNTIME_ERROR:
	Set path using:
		export LD_LIBRARY_PATH=/home/ahkamal/anaconda3/envs/open-mmlab/lib/python3.7/site-packages/torch/lib:$LD_LIBRARY_PATH


Steps for Singularity:

- Install Anaconda following directions from:
https://docs.anaconda.com/anaconda/install/linux/

- Make sure to initialise Anaconda in /home (not /.conda) using: (needs to be done everytime terminal is launched)
	source /home/ahkamal/anaconda3/bin/activate

- Create anaconda environment in this directory using:
	conda create -p /home/ahkamal/anaconda3/envs/open-mmlab python=3.7 -y
